Unified for Loops

Michael MacLeod

 (require unified-for) package: unified-for

This package unifies the various flavors of for iteration constructs—for, for/list, for/vector—into a single for macro.

syntax

(for maybe-accumulator (for-clause ...) body ...)

  | accumulator-expr
  | (accumulator-expr arg-expr ...)
